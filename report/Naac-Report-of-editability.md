# NAAC Department Editability Report

## Student Satisfaction Survey (SSS)
- **Editable Content**: Google Form URL/link
- **Character Limits**: URL/link (max 250 chars)
- **Item Limits**: Maximum 1 Google form link
- **Reason**: The survey target link changes periodically. Restricting it to just the URL ensures the form remains accurate without needing complex backend text management. 

## SSS Report
- **Editable Content**: Academic Year, PDF uploading section
- **Character Limits**: Academic Year (max 20 chars)
- **Item Limits**: Maximum 10 reports
- **Reason**: Maintains a clean historical record layout for reports without overwhelming the page with endless rows. PDF file (main document content – edited separately and replaced).

## SSR Cycle 1
- **Editable Content**: Criteria Headings, Metric Numbers, PDF uploading section
- **Character Limits**: Criteria Headings (max 150 chars), Metric Numbers (max 15 chars)
- **Item Limits**: Maximum 30 metric rows per category
- **Reason**: Keeps the extremely large volume of metric data cleanly organized and readable without breaking the page layout. PDF file (main document content – edited separately and replaced).

## SSR Cycle 2

### 1. Extended Profile
- **Editable Content**: Category Name (e.g., Students, Teachers), Metric IDs, PDF uploading section
- **Character Limits**: Category Name (max 50 chars), Metric IDs (max 15 chars)
- **Item Limits**: Maximum 15 rows depending on metrics
- **Reason**: Ensures accurate tracking of fundamental institutional data. PDF file (main document content – edited separately and replaced).

### 2. Quality Indicator Framework (QIF)
- **Editable Content**: Criteria Headers (e.g., Curricular Aspects, Teaching-Learning), Sub-Criteria Codes (e.g., 1.1.1), Short Description, PDF uploading section
- **Character Limits**: Criteria Name (max 150 chars), Sub-Criteria Code (max 15 chars)
- **Item Limits**: Maximum 30 metric rows per Criteria Category
- **Reason**: QIF is the largest section of NAAC. It must be strictly restricted to text fields and PDF uploading sections to prevent messy, broken layouts across the 7 demanding criteria blocks. PDF file (main document content – edited separately and replaced).

### 3. Data Validation and Verification (DVV) Clarifications
- **Editable Content**: Metric Code, Criteria Heading, PDF uploading section
- **Character Limits**: Metric Code (max 15 chars), Criteria Heading (max 150 chars)
- **Item Limits**: Maximum 50 clarification links overall
- **Reason**: DVV queries require structured responses. Hard-coded limits keep the list readable as users cross-reference the original SSR. PDF file (main document content – edited separately and replaced).

## Best Practices and Institutional Distinctiveness
- **Editable Content**: Practice Title, Short Description, PDF uploading section
- **Character Limits**: Practice Title (max 100 chars), Short Description (max 300 chars)
- **Item Limits**: Maximum 3 best practices
- **Reason**: Ensures clear, concise focus on the most important institutional strengths without making the text paragraphs too massive. PDF file (main document content – edited separately and replaced).
